• The fetch API allows JS to access and manipulate parts of the HTTP pipeline, such as requests and responses.
  • It allows fetch() - a logical way to grab resources asynchronously
  • It's different from jquery.ajax in these ways: the promise returned won't reject on HTTP error status, won't send cross origin cookies.
  • Fetch basic example:
  • fetch('http://example.com/movies.json')
    .then(response => response.json()
    ) .then(data => console.log(data));
  • The most basic fetch method just takes a paramater of the requested URL.
  • a second init paramter can be included that provides options to change various aspects.
  • See fetch() for the full options available, and more details.
