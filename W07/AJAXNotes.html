Ajax

  • Ajax: Ajax is a technique that allows web pages to communicate asynchronously with a server, and it dynamically updates web pages without reloading. This enables data to be sent and received in the background, as well as portions of a page to be updated in response to user events, while the rest of the program continues to run.
  • Ajax requires a server to run. because the point of Ajax is to gather webpages from another source (server).
  • AJAX:Asynchronous JavaScript and XML (although these days JSON is preferred over XML)
