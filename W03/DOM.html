Document Object Model

"document." is a built in object.

getElementById() and others are methods.

In the DOM, white space is an empty text nodes.

body.nodetype will shows what type of node object the body is.

Code Type
1 Element
2 attribute
3 text
8 Comment
9 body

There are other node types, but they aren't used in HTML.

Node Lists are array like, but they are not arrays.

node lists use index notation and the length method, but no other methods that arrays have.

simply use the Array.from() method to turn the node list into an array. Or the spread operator.([...document.images];)

getElementById, getElementsByTagName, GetElementsByClassName

querySelector() - first element that matches the css selector, and QuerySelectorAll() - returns node list of all elements with css selector

query selector methods can be used on any element. "ul.querySelector('li#bats')
