Document Object Model

"document." is a built in object.

getElementById() and others are methods.

In the DOM, white space is an empty text nodes.

body.nodetype will shows what type of node object the body is.

Code Type
1 Element
2 attribute
3 text
8 Comment
9 body

There are other node types, but they aren't used in HTML.

Node Lists are array like, but they are not arrays.

node lists use index notation and the length method, but no other methods that arrays have.

simply use the Array.from() method to turn the node list into an array. Or the spread operator.([...document.images];)

getElementById, getElementsByTagName, GetElementsByClassName

querySelector() - first element that matches the css selector, and QuerySelectorAll() - returns node list of all elements with css selector

query selector methods can be used on any element. "ul.querySelector('li#bats')

.childNodes returns all nodes, includeing empty text nodes. .Children returns ony children, meaning no empty text nodes.

There are many other methodxs to grab various nodes. be aware that many grab empties

one can use dot notation to access attributes.

in dot notation call and for need to be className and htmlFor

classlist.add to avoid replacing all classes.

Updating css

superman.style.border this is how to update css. If the css has a hyehn in its name, instead type it in camel case and it will work.

or like this: superman.style['background color']

to get a read-only list of the styles applied to an item use: getComputedStyle(). There are methods to help filter it.

CHrome and other browsers don't allow the use of some methods. May cause an error. probably don't use it in any actual code.

best pracrice is to either adjust the style dynamically or to update hte elements class and then add the class to a stylesheet.
